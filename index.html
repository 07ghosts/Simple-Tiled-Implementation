Simple Tiled Implementation

A powerful Tiled library made for the *awesome* LÖVE framework

View Documentation

Simple Tiled Implementation is feature rich and simple to use, just export your map from Tiled to Lua and import it directly into your game using STI.

  • Orthogonal Maps

  • Tile Layers
  • Object Layers

  • Layer Visibility
  • Layer Parallax Offsets
  • Layer Removal
  • Tile Offsets
  • Tile Flags

  • Update Callback
