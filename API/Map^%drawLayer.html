Map:drawLayer

Saturday, May 31, 2014 2:21 AM

If you need to draw an individual Layer, you can use this function.

Arguments

index

  • Name or Index (from bottom) of layer

Returns

nil

local sti = require("sti")

function love.load()
    map = sti.new("assets/maps/map01")
end

function love.draw()
    -- Draw using layer name
    map:drawLayer("Sprite Layer")

    -- Draw using layer index
    map:drawLayer(2)
end
