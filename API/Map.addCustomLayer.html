Map.addCustomLayer

Saturday, May 31, 2014 12:46 AM

If you need to include extra data in your map that Tiled does not support, such as a player object, fear not! You can create a custom layer and insert it anywhere in your map so that other layers can be drawn above or below it.

Arguments

self

  • Map object

name

  • Name of layer

index

  • Index (from bottom) of layer

Returns

nil

local sti = require("sti")

function love.load()
    map = sti.new("assets/maps/map01")

    map:addCustomLayer("Sprite Layer", 2)
end
