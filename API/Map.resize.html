Map.resize

Saturday, May 31, 2014 12:31 AM

If your game window is of a fixed size (such as on a mobile device), you can omit this function. If you plan to allow your user to change the window size at all, for any reason, this function is absolutely required!

Arguments

self

  • Map object

width

  • width of window

height

  • height of window

Returns

nil

local sti = require("sti")

function love.load()
    map = sti.new("assets/maps/map01")
end

function love.update(dt)
    map:update(dt)
end

function love.draw()
    map:draw()
end

function love.resize(width, height)
    map:resize(width, height)
end
